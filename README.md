# マクロ経済シミュレーション

## 概要

このプロジェクトは、マクロ経済データを用いて経済モデルの分析を行うものです。

## データ分析の流れ

1.  **データ加工**
    *   **使用データ**:
        *   `gdp_data_251214.csv`
        *   `longrate_data_251214.csv`
        *   `shortrate_data_251214.csv`
        *   `tax_data_251215.csv`
    *   **加工用ノートブック**: `test.ipynb`
    *   **加工後データ**: `d_data.csv`

    上記4つの元データを `test.ipynb` で加工し、分析用の `d_data.csv` を作成します。

2.  **モデル分析**
    *   **初期分析 (二段階最小二乗法)**
        *   **ノートブック**: `2LSL_sim.ipynb`
        *   当初、二段階最小二乗法による分析を試みましたが、納得のいく結果は得られませんでした。

    *   **最終分析 (誤差修正モデル)**
        *   **ノートブック**: `ECM.ipynb`
        *   次に、誤差修正モデル（ECM）による分析を行い、有意な結果が得られました。この `ECM.ipynb` が本プロジェクトの最終的な分析結果となります。

## ファイル構成

*   `test.ipynb`: データ加工用ノートブック
*   `2LSL_sim.ipynb`: 初期分析（二段階最小二乗法）のノートブック
*   `ECM.ipynb`: 最終分析（誤差修正モデル）のノートブック
*   `d_data.csv`: 加工済みデータ
*   `gdp_data_251214.csv`, `longrate_data_251214.csv`, `shortrate_data_251214.csv`, `tax_data_251215.csv`: 元データ
*   `Untitled.ipynb`, `data -20251031ver.csv`: 分析の試作ファイル（本筋とは無関係）
*   `tax_data_251214.csv`: 旧税データ（分析には不使用）
